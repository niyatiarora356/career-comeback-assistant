# Career Comeback Assistant

An ML-powered web app to help women re-enter the workforce after career gaps.

## Features (In Progress)
- Resume parsing with BERT NER
- Skill gap analysis
- Role recommendations using Sentence-BERT + FAISS
- Upskilling roadmap generation via LLM

## Tech Stack
- Python, HuggingFace Transformers, Sentence-BERT
- FAISS, spaCy, Streamlit, Gemini API
